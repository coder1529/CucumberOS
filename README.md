# 🥒 CucumberOS 🥒

## A CUSTOM OS!
- Custom OS with the theme of a cucumber
- Contains advanced note taker app
- Contains a Clicker game
- Contains : Background settings
- Contains : Terminal
- Contains : Screentime
- NEW! : Pickle jar
- NEW! : Add folders
- NEW! : Edit folders

## WHY A CUCUMBER?
- I really like the vegetable Cucumber
- It also sounds so good

## WHY IS THE NOTETAKER APP ADVANCED?
- Has a functional sidebar
- Has a functional color change for every new note
- Has a functional add a new note

## WHAT IS THE CLICKER GAME?
- It is a game called Cucumber Clicker
- Version 1.5
- Has a lot of items

## WHAT IS THE BACKGROUND SETTINGS?
- Allows you to customize the OS
- You can upload custom images
- Make the OS really yours!

## WHAT IS THE TERMINAL?
- Allows you to perform actions that might not be possible in the older version of the OS
- Has custom commands
- Commands:
  - /help: Gives you all the possible commands
  - /killAll: Kills all windows
  - /open {user_input}: Opens windows you want
  - /openAll: Opens all windows
  - /endOS: Exits you out of the OS

## WHAT IS THE SCREEN TIME APP?
- Contains monitoring powers
- Monitors every single app
- Shows you all the total time you have been on the os

## WHAT IS THE PICKLE JAR APP?
- Contains applications from the desktop
- Contains linked files with the advanced notetaker app
- Contains linked folders from the desktop

## WHAT IS THE ADD FOLDERS SKILL?
- You can add folders to the desktop
- Custom right click event
- Creates customizable folders

## WHAT IS THE EDIT FOLDERS SKILL?
- Allows you to change names of the folders
- Allows you to change icons of the folders
- Custom right click event
- Changes names in the desktop folders too
- Changes icons in the desktop folders too
